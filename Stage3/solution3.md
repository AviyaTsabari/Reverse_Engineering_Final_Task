נחפש את נקודת הפתיחה:

jump->jump to entry point 

ומצאנו שכתובת הכניסה היא: 01003E21.

כדי ליצור הזרקת קוד נצטרך section שבו אפשר להזריק את הקוד. לצורך כך נשתמש בCFF.

נטען את קובץ המשחק לתוכנה ושם בלשונית sections נלחץ קליק ימני בטבלה ונבחר add section וצריך לבחור גודל (הצאט אמר ש1000 זה מספיק).

נשנה את הדגלים של הסקשן כך שהוא יכיל קוד, ויהיה אפשר לכתוב לקרוא ולהריץ:

<img width="401" height="407" alt="image" src="https://github.com/user-attachments/assets/e674cb15-4994-488d-bb90-01f3a7e78baa" />

בנוסף, אנחנו רוצים לדעת מה הכתובת הוירטואלית של הסקשן, לכן נרצה imagebase שהוא1000000 והכתובת הוירטואלית של הסקשן זה 20000 אז הקוד החדש יהיה ב1020000.

נשמור בצד את הפקודות הראשונטת בתוכנית בשביל שנוכל לדרוס אותן ונשמור את הקובץ החדש יחד עם הסקשן שנוצר.

מצאתי ייבוא של הפונקציה MessageBoxW והכתובת ב010010B8.

קודם כל נדרוס את שתי הפקודות הראשונות ונכתוב קפיצה לסקשן החדש עם שתי פקודות Nop לאיזון.

עכשיו ניכנס לסקשן ונכתוב שם את הקוד. 

לא נותן לי לכתוב קוד אז נכנסתי לתצוגת הקסה ושם הכנסתי סתם ערכים זמניים 90 90 90.... וזה הכניס שורות ואת השורות האלה אני אפצפץ!

זה הקוד שהכנסנו:

    pushfd ;להחזיר מצב קודם לפני קריאה לפונקציות של ווינדוס
  
    pushad


    push    0
   
    push    01020080h          ; כתובת הכותרת
   
    push    01020040h          ; כתובת תוכן ההודעה
   
    push    0
 
    call    dword ptr [010010B8h] ; MessageBoxW

    popad
  
    popfd
    
    push    70h
   
    push    01001390h          ; הפקודות המקוריות שנדרסו
   
    jmp     01003E28h          ; חזרה לקוד המקור
